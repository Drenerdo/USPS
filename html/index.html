Command line test usage

When this test application is run from the command line, tests of the API can be selected. A full amplitude sine wave audio at 261.6 hz is always output.

Backlight

            ./test backlight [brightness [grayscale]]

Set backlight brightness (0-127) and grayscale (0-65535).

Backlight Blink

            ./test backlightBlink [onMS [offMS [count]]]

Set backlight blink. onMS and offMS default 1000, count default is 3.

Backlight PulseGS

            ./test backlightPulseGS [durationMS]

Pulse backlight grayscale 5 times. Default durationMS is 1000.

Backlight Pulse

            ./test backlightPulse [durationMS]

Pulse backlight brightness 5 times. Default durationMS is 1000.

Dotstar

            ./test dotstar [test]

Test the LED strip. Default test is 0.

  • 0 fade brightness
  • 1 rotate

Display Init

            ./test displayInit

Clears the text. Must be called before other text commands.

Display Text

            ./test displayText [text]

Display the text. The default text is "A.B.C.D.1.2.3.4".

Display Brightness

            ./test displayBrightness {brightness1 [brightness2]]

Set the brightness of the text. Brightness1 left half, brightness2 right half.

Display Blink

            ./test displayBlink [rate1 [rate2]]

Blink the text.

  • 0 on, no blink
  • 1 on, 2 Hz blink
  • 2 on, 1 Hz blink
  • 3 on, 0.5 Hz blink
  • 4 off

API

usps_bb.api.h and usps_bb_api.c contains the API to be used by the application.
